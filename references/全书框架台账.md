# 全书框架台账（五类提炼）

基于《创新力：从思维到能力的企业增长之路》（混沌学园，2021）全文提炼。

## 一、框架（核心必进 ★★★）

### 个人创新力

| 名称（保留原名） | 一句话定义 | 何时用 | 原书表述 |
|----------------|-----------|--------|---------|
| **创新力模型** | 沿心智四维拉伸，由组合/进化/差异/破界四大思维构成 | 诊断创新能力短板 | "沿人类理性推理心智的拉伸方向，由四个思维能力组成" |
| **组合思维能力** | 拆解要素→抓关键单一要素→重构价值组合 | 面对复杂问题需要突破口 | "揭开表面，拆解要素，抓重点，重构价值组合" |
| **进化思维能力** | 沿时间维度动态思考，识别趋势与时机 | 判断发展阶段、找第二曲线 | "超越静态，随时间向前看，动态分析趋势" |
| **差异思维能力** | 反从众、向边缘看、找差异化生态位 | 红海竞争需要突围 | "破除常规，反从众反主流向外看，识别边缘空间" |
| **破界思维能力** | 沿纯逻辑维度深挖，打破隐含假设 | "不可能"式困局需要根本突破 | "打破原有边界，产生颠覆性成果" |
| **三层次创新实践** | 创新成果（表）→创新行为（中）→创新思维（核） | 区分伪创新和真创新 | "创新思维是最核心的，是创新价值的来源" |

### 组织创新力

| 名称（保留原名） | 一句话定义 | 说明 |

## 二、原则

| 名称 | 一句话定义 | 原书表述 |
|------|-----------|---------|
| **生命以负熵为生** | 一切有机体的底层原理：持续熵减增加确定性 | "生命以负熵为生"——薛定谔 |
| **创新=有价值的改变** | 不是所有改变都是创新，必须有增量价值 | "创新，是有价值的改变，change for good" |
| **创新=旧要素新组合** | 创新不是无中生有，是已有要素创造性重组 | "创新，是旧要素的新组合"——熊彼特 |
| **创新是有方向的不随机** | 创新不靠头脑风暴和灵感，有心智拉伸方向 | "创新是有方向的，不是随机的" |
| **创新力=转化不确定性** | 创新力就是处理不确定性→确定性的能力 | "创新力即应对不确定性转化为确定性的能力" |
| **守正出奇** | 第一曲线稳健管理，第二曲线积极进取 | "守正出奇：企业要对第一曲线进行管理，对第二曲线积极进取" |
| **独立小机构** | 新业务必须与原有组织切割，建立独立价值网 | "创V组织必须与原有组织体系隔离开来" |

## 三、技法

| 名称 | 操作 | 原书表述 |
|------|------|---------|
| **定义真问题=目标+障碍** | 先追问初心目标，再拆解障碍 | "真问题=目标+障碍" |
| **要素拆解到最小可控单元** | 复杂问题拆成可直接操作的小块 | "将障碍进行要素拆解，寻找到最小可控价值单元" |
| **找十倍速变化要素** | 识别环境中最快变化的要素，借势发力 | "找到十倍速变化的趋势要素借势而为" |
| **饱和攻击/力出一孔** | 找到单一要素后所有资源砸上去 | "找一个最小切口，把精兵强将、资源调上去，饱和攻击" |
| **S曲线诊断** | 判断业务处于S曲线哪个阶段决定策略 | "任何品类的增长形态都是S曲线" |
| **孵化机制** | 给创新项目提供资源+服务+空间 | "通过生物式孵化机制提升应对不确定性的能力" |
| **心智拉伸四维法** | 有意识沿物质/时间/空间/纯逻辑扩展 | "演绎思维有四个维度：要素、时间、空间和纯逻辑" |

## 四、反模式

| 名称 | 是什么 | 原书警示 |
|------|--------|---------|
| **乱创新=找死** | 无价值方向的随机改变 | "不创新是等死，但不要因为乱创新而找死" |
| **盲目头脑风暴** | 迷信发散创意数量=创新 | "低质量创意越多，浪费资源越多" |
| **迷信感性=创新** | 以为灵感、感性的人更善于创新 | "大部分创新成功的人是理性的人" |
| **木桶理论陷阱** | 执着补短板不敢打长板 | "取胜的系统往往在最大化单一要素上走到荒谬的极端" |
| **大组织绞杀新业务** | 用老组织流程心智管新业务 | "在位企业输给内部的既得利益和既有思维" |
| **静态思维** | 以为现在如此将来也如此 | "传统管理的错觉：静态、线性、连续" |
| **多元化=伪创新** | 脱离主业胡乱扩张 | "多元化布局已被市场证明不是科学商业模式" |

## 五、作者声音

- **底层逻辑倾向**：极致追求"第一性原理"，不断溯源生命科学/物理学
- **学术味+实用**：大量引经典（熊彼特、克里斯坦森、平克、香农），但比教科书实用
- **金句式表达**："走进混沌是为了走出混沌""一个人是否青春在于还能不能处理不确定性"
- **混沌学园术语**：分形、击穿阈值、奇点下移、第一/第二曲线
- **理性崇拜**：不推崇感性和直觉，推崇逻辑推理和独立思考
