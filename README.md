# 创新力：从思维到能力的企业增长之路

## 关于这个 Skill

基于混沌学园《创新力：从思维到能力的企业增长之路》（中信出版社，2021，张雷主编）提炼的 AI 思维框架。

把这本书 19 万字提炼成一整套 AI 可以直接调用的创新工具：
- **4 个创新思维能力** + 对应思维模型
- **T-RPC 组织建设框架** + 实践案例
- **3 种创新型组织形态** + 选择指南

---

## 🚀 安装

**第一步：下载**
```bash
git clone https://github.com/heartbreaker5970-dotcom/chuangxinli-skill.git
```
没有 git？直接点页面上的绿色 Code 按钮 → Download ZIP 解压。

**第二步：看你用什么 AI 工具**

| 你的工具 | 怎么用 |
|---------|--------|
| **ChatGPT / Claude / Kimi / DeepSeek / 豆包 / 元宝** | 打开 `system-prompt.md`，全文复制粘贴到 system prompt / 自定义指令 |
| **支持上传文件的 AI** | 把整个文件夹拖进去，说「加载这些文件，用书里的框架回答创新问题」 |
| **EasyClaw** | 把 `chuangxinli/` 整个文件夹放到 `workspace/skills/` 目录，重启 |

**就两步，完了。**

---

## 📂 文件说明

| 文件 | 内容 |
|------|------|
| `system-prompt.md` | **核心提示词**，直接复制到 AI 的 system prompt |
| `references/创新力模型四维能力.md` | 组合思维/进化思维/差异思维/破界思维 |
| `references/创新型组织T-RPC模型.md` | 人才/资源/机制/文化四维 |
| `references/进化思维与第二曲线.md` | S曲线、分形创新、守正出奇 |
| `references/组合思维与单点破局.md` | 要素拆解、单一要素饱和攻击 |
| `references/差异思维与错位竞争.md` | 错位竞争、低端颠覆 |
| `references/破界思维.md` | 隐含假设、逻辑奇点下移 |
| `references/三大创新型组织形态.md` | 创E/创V/创X |
| `references/创新本质与基本原则.md` | 负熵、三层次、原则反模式 |
| `references/全书框架台账.md` | 完整提炼总表 |

## 📖 适用场景

- 想做第二曲线但不知道怎么找
- 跟大厂竞争不知道怎么突围
- 团队死气沉沉想激活创新能力
- 产品/业务需要创新但无从下手
- 想系统学习创新方法论

## ⚠️ 使用建议

- ✅ **先加载 system-prompt.md**，让 AI 知道这套框架
- ✅ **按需加载 references**，不要一次性全丢
- ✅ **用日常话提问**，比如"我这业务增长不动了怎么办"，AI 会自动调用对应框架
- ❌ 不需要先通读全书再使用
- ❌ 不需要理解所有概念才开始

## 📦 一键安装（EasyClaw）

```bash
git clone https://github.com/heartbreaker5970-dotcom/chuangxinli-skill.git && mv chuangxinli-skill skills/chuangxinli
```
然后重启 EasyClaw。
