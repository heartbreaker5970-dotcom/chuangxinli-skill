# 创新力：从思维到能力的企业增长之路

基于混沌学园《创新力：从思维到能力的企业增长之路》（中信出版社，2021）提炼的思维框架与组织建设工具。作者张雷主编，混沌创新力学习中心出品。

## 核心论点

> **创新是有方向的，不是随机的。创新力 = 应对不确定性、将其转化为确定性的能力。** 大多数人把创新归于灵感、头脑风暴和随机运气——这是错的。真正成功的创新来自四个方向的思维能力，这些能力有模型可学、有章法可循。

## 当用户说这些话时，你应该想起我

- "我想找到新的增长点" / "业务遇到天花板了"
- "怎么跟巨头竞争" / "大家都在做一样的事怎么突围"
- "这个商业模式怎么创新" / "我们的产品需要差异化"

- "这个新业务要不要做" / "什么时候该开辟新业务"
- "觉得自己思维有局限，跳不出来"

## 基础速查：创新力模型四个方向

1. **组合思维** → 洞察价值：拆解要素 → 抓关键单一要素 → 饱和攻击击穿阈值（参考：马斯克、海底捞服务单点、苹果iPod）
2. **进化思维** → 洞察时间：S曲线判断阶段 → 识别趋势 → 分形长出第二曲线（参考：马云阿里云、小米新国货）
3. **差异思维** → 洞察空间：反从众 → 找边缘 → 错位竞争（参考：美团二线城市、拼多多下沉市场）
4. **破界思维** → 洞察本质：打破隐含假设 → 逻辑奇点下移 → 重构系统

## 核心公式速记

- **创新** = 有价值的改变 = 旧要素的新组合
- **真问题** = 目标 + 障碍
- **创新力** = 处理不确定性 → 转化为确定性
- **创新成功** = 单点突破 + 饱和攻击 → 击穿阈值
- **创新型组织** = 外在适应力 + 内在活力
- **第二曲线** = 第一曲线 × 分形创新 + 独立小机构

## 重要提醒

- 创新不是头脑风暴、不靠灵感和天才——靠思维能力，可习得
- 新业务必须独立小机构，不能放在老组织里养
- 资源聚焦单一要素饱和攻击，不要撒胡椒面
- "与其更好，不如不同"——不要跟巨头在同一维度硬碰
